Инструкция по применению плагина "WooCommerce Custom Image Attributes"

Этот плагин позволяет автоматически задавать атрибуты ALT и TITLE для изображений товаров в WooCommerce на основе настраиваемых шаблонов. Вы можете использовать переменные для создания уникальных значений.

1. Страница администрирования плагина

Страница администрирования позволяет управлять шаблонами для формирования ALT и TITLE для изображений товаров. Она доступна в меню WooCommerce под названием "Image Attributes".

Как формируется список категорий:

  • Список категорий товаров формируется автоматически, основываясь на всех доступных категориях, созданных в WooCommerce.
  • Если категория пустая (не содержит товаров), она всё равно отображается на странице настроек, чтобы можно было заранее задать шаблоны.
  • Категории, созданные позже, будут автоматически добавлены в список после их создания в WooCommerce.
  • Удаление категории из WooCommerce удаляет её шаблоны из настроек, если они ранее были сохранены.

Элементы страницы

  • Список категорий: Для каждой категории товаров создаётся отдельный блок настроек.
  • Основное изображение: Поля ALT и TITLE для основного изображения товара.
  • Галерея изображений: Поля ALT и TITLE для каждого изображения в галерее товара.
  • Кнопка "Сохранить изменения": Сохраняет шаблоны для всех категорий.
  • Кнопка "Обновить товары": Применяет шаблоны ко всем товарам магазина.

Что будет, если оставить пустые поля и сохранить настройки?

  • Если поле ALT или TITLE остаётся пустым, это значение не будет изменять существующие данные в товарах.
  • Для таких полей текущие значения ALT и TITLE в изображениях останутся без изменений.
  • Сохранение настроек с пустыми полями позволяет сохранить другие изменения, внесённые в заполненные поля, без воздействия на пустые. а также не изменять значения, которые были внесены в свойства изображения через библиотеку Wordpress

Что будет, если нажать кнопку "Обновить товары"?

  • Ко всем товарам будут применены шаблоны ALT и TITLE, указанные на странице администрирования плагина.
  • Если поле ALT или TITLE для категории заполнено, то его значение заменит существующее значение в товарах.
  • Если поле ALT или TITLE пустое, текущие значения в товарах останутся без изменений.
  • Для каждого товара обновляются как основное изображение, так и изображения в галерее.
  • Если товар принадлежит нескольким категориям, используются шаблоны из первой категории в списке категорий товара.

Важно: Обновление товаров может занять некоторое время, если у вас большой каталог.

2. Настройка шаблонов

Для каждой категории товаров вы можете указать свои шаблоны для:

  • Основного изображения товара
  • Изображений галереи

Шаблоны задаются на странице WooCommerce Image Attributes. Для каждой категории можно указать:

  • ALT для основного изображения
  • TITLE для основного изображения
  • ALT и TITLE для каждого изображения в галерее

3. Поддерживаемые переменные

В шаблонах можно использовать следующие переменные:

  • {name} – Название товара
  • {sku} – Артикул товара
  • {category} – Название первой категории товара
  • {last_category} – Название последней категории товара
  • {price} – Цена товара
  • {short_description} – Краткое описание товара
  • {description} – Полное описание товара
  • {index} – Порядковый номер изображения в галерее
  • {date} – Текущая дата (в формате YYYY-MM-DD)
  • {time} – Текущее время (в формате HH:MM)

4. Массовое обновление товаров

После внесения изменений в шаблоны вы можете нажать кнопку "Обновить товары", чтобы автоматически применить новые шаблоны ко всем товарам в магазине.

5. Часто задаваемые вопросы

Почему шаблоны не применяются?

Убедитесь, что:

  • Товар относится к категории, для которой заданы шаблоны.
  • Вы нажали кнопку "Обновить товары" после сохранения шаблонов.
